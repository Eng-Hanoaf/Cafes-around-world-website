# 🌍 Cafes Around The World

A comprehensive, multi-page tourism and marketing website showcasing famous global cafes (including EL&N London, De Flora Paris, Ulus Turkey, and Haraz Yemen). Built completely from scratch to master structure, navigation, and multimedia integration.

## ✨ Features
* **Multi-Page Navigation:** Seamless architecture linking the home page with individual cafe dynamic profiles and a dedicated Login system.
* **Rich Multimedia Integration:** Features beautifully embedded video tours and interactive media components to enhance storytelling.
* **Modern Dark Aesthetics:** Designed with a sleek, content-focused dark theme that elevates visual presentation.
* **Clean & Semantic Code:** Formatted with organized styles and standard layouts for structural efficiency.

## 🛠️ Technologies Used
* **HTML5:** For comprehensive document structure, forms, and video/audio components.
* **CSS3:** For complete layouts, responsive typography, visual presentation, and smooth hover transitions.
